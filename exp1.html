To implement an exhaustive search algorithm for the Tic-Tac-Toe game using Python to explore all possible game states and determine the optimal outcome

AIM

To implement an exhaustive search algorithm for the Tic-Tac-Toe game using Python to explore all possible game states and determine the optimal outcome.

ALGORITHM

  1. Start.
  2. Initialize a 3×3 Tic-Tac-Toe board with empty spaces.
  3. Display the board and position indices (0–8).
  4. Define a function print_board to display the grid in rows with separators.
  5. Define a function check_winner to return 'X' or 'O' if any winning combination matches; else return None.
  6. Define a function to check whether the board is full.
  7. Implement Exhaustive Search (Minimax): Base Cases: 'X' wins → +1, 'O' wins → −1, Draw → 0. Recursive Cases: Maximizing (AI = X): Try all empty cells → place 'X' → call minimax(False) → undo move → keep maximum score. Minimizing (Human = O): Try all empty cells → place 'O' → call minimax(True) → undo move → keep minimum score. Evaluate all empty cells and choose the move with the highest score.
  8. Alternate turns between AI and human until a win or draw occurs.
  9. After each move, check for a winner or draw condition.
  10. If a winner or draw is found, display the result and stop the game.
  11. End.

SOURCE CODE

board = [' ' for _ in range(9)]
def print_board():
    print()
    for i in range(3):
        print("|".join(board[i*3:(i+1)*3]))
        if i<2:
            print(" ------- ")
    print()
def check_winner(b):
    wins=[(0,1,2),(3,4,5),(6,7,8),(0,3,6),(1,4,7),(2,5,8),(0,4,8),(2,4,6)]
    for a,b1,c in wins:
        if b[a] == b[b1] == b[c] and b[a]!=' ':
            return b[a]
    return None
def board_full(b):
    return ' 'not in b
def exhaustive_search(b ,player):
    if check_winner(b) or board_full(b):
        return
    for i in range(9):
        if b[i]== ' ':
            b[i]=player
            exhaustive_search(b,'0' if player == 'x' else 'x')
            b[i]= ' '
def ai_move():
    for i in range(9):
        if board[i] == ' ':
            board[i] = 'X'
            exhaustive_search(board,'0')
            return i
print("TIC-TAC-TOE")
print("AI =X | human=0")
print("positions:")
print(" 0 1 2 \n3 4 5\n6 7 8")

print_board()

while True:
    move = ai_move()
    board[move] = 'x'
    print("AI played:",move)
    print_board()

    if check_winner(board):
        print("winner:",check_winner(board))
        break
    if board_full(board):
        print("game draw")
        break
        
    pos=int(input("your move (0 -8): "))
    if board[pos] !=' ':
        print("invalid move. try again.")
        continue
    board[pos]='0'
    print_board()

    if check_winner(board):
        print("winner:",check_winner(board))
        break
    if board_full(board):
        print("game draw")
        break
