To implement the Breadth-First Search (BFS) algorithm for solving the Water Jug Problem

AIM

To implement Breadth First Search (BFS) and Depth First Search (DFS) to solve the Water Jug problem and reach the target state.

ALGORITHM

  1. Start the program.
  2. Define jug capacities (A, B) and target value.
  3. Represent state as (x, y) where x and y are water in jugs.
  4. Initialize initial state (0, 0).
  5. For BFS → use queue, for DFS → use stack/recursion.
  6. Generate all possible next states: Fill A, Fill B, Empty A, Empty B, Pour A → B, Pour B → A.
  7. Check if any state equals target → stop.
  8. Otherwise, continue until solution is found or all states are explored.

PROGRAM (BFS)

from collections import deque
def bfs(a, b, target):
    visited = set()
    q = deque([(0, 0)])
    while q:
        x, y = q.popleft()
        print((x, y))
        if x == target or y == target:
            return "Target Reached"
        if (x, y) in visited:
            continue
        visited.add((x, y))
        q.extend([
            (a, y), (x, b),
            (0, y), (x, 0),
            (max(0, x -(b-y)), min(b, x+y)),
            (min(a, x+y), max(0, y -(a-x)))
        ])
    return "Not Possible"
print(bfs(4, 3, 2))

PROGRAM (DFS)

def dfs(x, y, a, b, target, visited):
    if (x, y) in visited:
        return False
    print((x, y))
    if x == target or y == target:
        return True
    visited.add((x, y))
    return (dfs(a, y, a, b, target, visited) or
            dfs(x, b, a, b, target, visited) or
            dfs(0, y, a, b, target, visited) or
            dfs(x, 0, a, b, target, visited) or
            dfs(max(0, x -(b-y)), min(b, x+y), a, b, target, visited) or
            dfs(min(a, x+y), max(0, y -(a-x)), a, b, target, visited))
print(dfs(0, 0, 4, 3, 2, set()))
