To implement the A* search algorithm for pathfinding in a grid-based environment

AIM

To implement the A* algorithm to find the shortest path between source and destination using heuristic function.

ALGORITHM

  1. Start the program.
  2. Initialize open list (nodes to visit) and closed list (visited nodes).
  3. Add the start node to open list.
  4. Assign g(n)=0 and calculate f(n)=g(n)+h(n).
  5. Select node with lowest f(n) from open list.
  6. If selected node is goal → stop and print path.
  7. Otherwise, move node to closed list and expand its neighbors.
  8. Update g(n), h(n), f(n) for neighbors and repeat until goal is reached.

PROGRAM

def astar(graph, h, start, goal):
    open_list = [start]
    closed = []
    g = {start: 0}
    parent = {start: None}
    while open_list:
        n = min(open_list, key=lambda x: g[x] + h[x])
        if n == goal:
            path = []
            while n:
                path.append(n)
                n = parent[n]
            return path[::-1], g[goal]   # return cost also
        open_list.remove(n)
        closed.append(n)
        for (m, cost) in graph[n]:
            if m not in open_list and m not in closed:
                open_list.append(m)
                parent[m] = n
                g[m] = g[n] + cost
            else:
                if g[m] > g[n] + cost:
                    g[m] = g[n] + cost
                    parent[m] = n
                    if m in closed:
                        closed.remove(m)
                        open_list.append(m)
    return None, None
graph = {
    'A': [('B', 1), ('C', 3)],
    'B': [('D', 1)],
    'C': [('D', 1)],
    'D': []
}
h = {'A': 4, 'B': 2, 'C': 1, 'D': 0}
path, cost = astar(graph, h, 'A', 'D')
print("Path:", path)
print("Cost:", cost)

OUTPUT

Path: ['A', 'B', 'D']
Cost: 2

RESULT

The A* algorithm was successfully implemented and the shortest path along with minimum cost was obtained.
